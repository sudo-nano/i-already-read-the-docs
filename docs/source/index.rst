I already read the docs, and they didn't help!
===================================

Sometimes you have a tech problem, and when you ask for help, people point you
to the docs. If you're being responsible and practicing :doc:`3 before me </3_before_me>`,
you probably already read the documentation, and it didn't help. That's why you're
asking them.
This site is a collection of resources on documentation, with examples of both
good and bad documentation as well as discussion on the topic.
Hopefully, it will help people write better documentation for their projects.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
